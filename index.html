You are given the following sales data. Complete all the tasks below:

sales_data = { 'Product': ['Laptop', 'Mouse', 'Keyboard', 'Monitor', 'Headphones', 'Laptop', 'Mouse'], 'Category': ['Electronics', 'Accessories', 'Accessories', 'Electronics', 'Accessories', 'Electronics', 'Accessories'], 'Price': [999.99, 25.50, 75.00, 299.99, 149.99, 1299.99, 29.99], 'Quantity': [2, 10, 5, 3, 8, 1, 15], 'Date': ['2024-01-15', '2024-01-16', '2024-01-16', '2024-01-17', '2024-01-17', '2024-01-18', '2024-01-18'] }

a) Create a DataFrame from the sales_data dictionary and display its structure. (5 marks)

b) Add a new column called 'Total_Sales' that calculates the total sales for each row (Price × Quantity). Display the updated DataFrame. (5 marks)

c) Find and display: (8 marks)

  • The product with the highest total sales
  • The average price of all products

Create a comprehensive student performance analysis system:

c) Perform the following analysis: (8 marks)

  • Find the student with the highest average score

d) Write code to create a summary report showing: (3 marks)
